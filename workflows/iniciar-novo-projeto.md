# 🆕 Workflow: Iniciar Novo Projeto

## Processo
1. **Definir stack** — React/Next.js? Laravel? PHP? Node?
2. **Criar estrutura** — pastas, arquivos base, config
3. **Aplicar design system** → `design-system-builder` + `color-typography-system`
4. **Configurar responsividade** → `responsive-mobile-first`
5. **Criar componentes base** → `component-upgrade` + `buttons-cta-specialist`
6. **Implementar layout** → `frontend-premium-ui`
7. **Adicionar dark mode** → via CSS custom properties
8. **Validar acessibilidade** → `accessibility-a11y-audit`
9. **Configurar segurança base** → `security-code-audit`
10. **Documentar** — README com setup e padrões
