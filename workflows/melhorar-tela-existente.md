# 🖥️ Workflow: Melhorar Tela Existente

## Processo
1. **Identificar a tela** — qual página/componente melhorar
2. **Detectar stack** — verificar como o CSS é gerenciado
3. **Auditar visual** → `frontend-redesign-audit`
4. **Listar problemas** — espaçamento, cores, tipografia, estados faltantes
5. **Aplicar design tokens** → `design-system-builder` (se não existirem)
6. **Melhorar componentes** → `component-upgrade`
7. **Melhorar formulários** → `forms-inputs-ux`
8. **Melhorar botões** → `buttons-cta-specialist`
9. **Adicionar animações** → `motion-animation-specialist`
10. **Validar responsividade** → `responsive-mobile-first`
11. **Validar acessibilidade** → `accessibility-a11y-audit`
12. **Entregar resumo** — arquivos alterados + antes/depois
