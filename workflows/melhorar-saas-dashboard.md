# 📊 Workflow: Melhorar SaaS Dashboard

## Processo
1. **Mapear dashboard** — todas as telas e componentes
2. **Detectar stack** — React, Next, Laravel, etc.
3. **Aplicar skill** → `dashboard-saas-ui`
4. **Melhorar sidebar** — navegação clara, ícones, collapse
5. **Melhorar header** — search, notifications, user menu
6. **Melhorar cards de métricas** — ícones, valores, variação
7. **Melhorar tabelas** — sort, filter, pagination, empty states
8. **Melhorar formulários** → `forms-inputs-ux`
9. **Adicionar design tokens** → `design-system-builder`
10. **Implementar dark mode**
11. **Validar responsividade** — sidebar colapsável no mobile
12. **Adicionar loading states** — skeletons para data fetching
13. **Validar acessibilidade** → `accessibility-a11y-audit`
