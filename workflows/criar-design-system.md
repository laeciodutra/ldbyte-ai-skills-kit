# 🎨 Workflow: Criar Design System

## Processo
1. **Auditar visual atual** → `frontend-redesign-audit`
2. **Definir paleta** → `color-typography-system`
3. **Definir tokens** → `design-system-builder`
4. **Implementar CSS custom properties** — cores, tipografia, espaçamento, sombras, radius
5. **Criar componentes base** — Button, Card, Input, Badge, Modal
6. **Implementar dark mode** — via `[data-theme="dark"]`
7. **Documentar tokens** — em arquivo acessível
8. **Migrar componentes existentes** — progressivamente
9. **Validar acessibilidade** → `accessibility-a11y-audit`
10. **Validar consistência** — todos os componentes usam tokens
