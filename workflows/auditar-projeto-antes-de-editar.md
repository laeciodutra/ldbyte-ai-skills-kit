# 🔍 Workflow: Auditar Projeto Antes de Editar

## Processo
1. **Mapear estrutura** — pastas, arquivos, dependências
2. **Detectar stack** — framework, CSS, componentes, build tool
3. **Avaliar código** → `full-project-review`
4. **Avaliar segurança** → `security-code-audit` (Fase 1 apenas — não corrigir)
5. **Avaliar UI** → `frontend-redesign-audit`
6. **Avaliar performance** → `performance-frontend-audit`
7. **Avaliar acessibilidade** → `accessibility-a11y-audit`
8. **Gerar relatório** com pontuação por categoria
9. **Priorizar melhorias** — crítico → importante → nice-to-have
10. **Apresentar plano** — esperar aprovação antes de editar
