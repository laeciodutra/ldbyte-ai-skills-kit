# 🔄 Workflow: Converter HTML/PHP para UI Moderna

## Processo
1. **Mapear projeto** — identificar todas as views/páginas
2. **Detectar stack PHP** → `php-laravel-ui-specialist`
3. **Auditar visual** → `frontend-redesign-audit`
4. **Criar design tokens CSS** → `design-system-builder`
5. **Substituir CSS inline** por classes organizadas
6. **Melhorar tipografia** — Google Fonts (Inter/Outfit)
7. **Melhorar paleta** — HSL com variáveis CSS
8. **Melhorar componentes** — botões, cards, forms, tables
9. **Adicionar responsividade** → `responsive-mobile-first`
10. **Adicionar dark mode** (se viável)
11. **Adicionar animações leves** — hover, transitions
12. **Validar** — todas as páginas funcionando
13. **NÃO trocar PHP por React/Vue** sem permissão
