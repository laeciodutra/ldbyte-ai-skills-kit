# Exemplo: Projeto Laravel + Blade

## Cenário
Projeto Laravel 11 com Blade templates, Tailwind CSS via Vite. Sistema administrativo com CRUD.

## Skills Aplicáveis
1. `php-laravel-ui-specialist` — UI em Blade
2. `design-system-builder` — Tokens CSS
3. `dashboard-saas-ui` — Dashboard admin

## Workflow Recomendado
`converter-html-php-para-ui-moderna.md`

## Comandos
```
Leia .ai-skills/skills/php-laravel-ui-specialist.md.
Melhore resources/views/admin/dashboard.blade.php.
Respeite @extends, @section, @component. NÃO altere rotas ou controllers.
```

## Resultado Esperado
- Layout Blade com design tokens CSS
- Componentes Blade (x-button, x-card, x-input)
- Dark mode via CSS custom properties
- Responsividade mobile-first
- @csrf mantido em todos os formulários
