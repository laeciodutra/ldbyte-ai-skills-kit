# Exemplo: Projeto React + Vite

## Cenário
Projeto React 18 com Vite, CSS Modules. Aplicação SPA com rotas, formulários e listagem de dados.

## Skills Aplicáveis
1. `react-next-ui-specialist` — Componentes React modernos
2. `design-system-builder` — Design tokens via CSS custom properties
3. `forms-inputs-ux` — Formulários com validação

## Workflow Recomendado
`melhorar-tela-existente.md`

## Comandos
```
Leia .ai-skills/skills/react-next-ui-specialist.md.
Aplique ao componente src/components/UserForm.tsx.
Melhore validação inline, estados visuais e feedback de submit.
```

## Resultado Esperado
- Formulário com validação inline
- Loading state no botão submit
- Error/success feedback com toast
- Responsividade mobile-first
