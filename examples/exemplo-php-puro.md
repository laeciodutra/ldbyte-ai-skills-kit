# Exemplo: Projeto PHP Puro

## Cenário
Sistema web em PHP puro com HTML/CSS/JS inline. Sem framework, sem build tool. Múltiplas páginas com includes.

## Skills Aplicáveis
1. `php-laravel-ui-specialist` — UI para PHP
2. `legacy-project-modernizer` — Modernização progressiva
3. `design-system-builder` — Tokens CSS

## Workflow Recomendado
`converter-html-php-para-ui-moderna.md`

## Comandos
```
Leia .ai-skills/skills/legacy-project-modernizer.md.
Modernize o visual de index.php progressivamente.
NÃO troque PHP por framework. Melhore HTML/CSS/JS apenas.
Substitua CSS inline por classes organizadas.
Adicione Google Fonts (Inter), paleta HSL e espaçamento consistente.
```

## Resultado Esperado
- CSS organizado em arquivo separado com custom properties
- Visual moderno sem trocar o stack
- Tipografia profissional
- Responsividade básica
- Lógica PHP 100% preservada
